## Installation

### Via Npm:

```
npm install car-names
```

### Usage

```javascript
var carNames = require('car-names');
var allNames = carNames.all;
var randomName = carNames.random();
```

### Via Browser:

```javascript
<script src="dist/index.umd.min.js"></script>
<script>
var allNames = carNames.all;
var randomName = carNames.random();
```
